# jekblog
A blog written in the Jekyll framework currently living at: [corylee.io](http://corylee.io)

### Requirements: 
Docker

### Windows commands:
#### To start with drafts included:
`$ ./start`

#### To build for production:
`$ ./publish`

### Linux commands:
#### To start with drafts included:
`$ ./linux-start`

#### To build for production:
`$ ./linux-publish`
